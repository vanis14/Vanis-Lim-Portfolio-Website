Olist ETL

This project makes use of multiple data sources from Olist, a Brazilian e-commerce website's sales data. The data is being transformed by using the OLAP and ETL methods in Tableau to combine and restructure the data. The staged data is then being stored in MySQL database until it is extracted by Talend and creates multiple reports to summarize the information. This project is completed as part of my “Business Intelligence and Data Warehousing” course's assessment.
